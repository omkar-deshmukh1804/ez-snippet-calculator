## Run Locally

Clone the project

```bash
  git clone https://github.com/omkar-deshmukh1804/ez-snippet-calculator
```

Go to the project directory

```bash
  cd ez-snippet-calculator
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
